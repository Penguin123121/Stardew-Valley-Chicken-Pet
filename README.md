# 🐤 星露谷小鸡桌宠

> 一只从星露谷物语（Stardew Valley）中走出来的白色小鸡，在你的 Windows 桌面上自由走动、发呆、和你互动 ✨

![Python](https://img.shields.io/badge/Python-3.7+-blue) ![PyQt5](https://img.shields.io/badge/PyQt5-5.15+-green) ![Windows](https://img.shields.io/badge/Windows-10%2F11-0078D6)

---

## ✨ 功能

| 操作 | 效果 |
|------|------|
| 🖱️ **点一下小鸡** | 头上冒 ❤️ 爱心 |
| 🖱️ **按住 2 秒** | 冒 💬 省略号 → 小鸡飞快逃跑 |
| 🖱️ **快速拖拽 1 秒** | 冒 😵 一团乱麻 |
| 🖱️ **右键托盘 → 🌿 喂食** | 放干草 → 小鸡走过去吃 → 😊 笑脸 |
| 🖱️ **拖拽小鸡** | 把它挪到你喜欢的位置 |
| 🖱️ **右键托盘菜单** | 设置大小/速度/活跃度/置顶/穿透/自启 |

小鸡会自己决定走路还是发呆，活跃度越高走路越多 🏃

## 📦 安装

### 一键安装（推荐）

双击 `install.bat`，自动完成：安装依赖 → 创建桌面快捷方式。

### 手动安装

```bash
# 1. 安装依赖
pip install PyQt5

# 2. 启动！
pythonw -m src.main
```

> **注意**：一定要用 `pythonw`（不是 `python`）启动，否则会弹出控制台黑窗口。

## 🎮 交互指南

### 和小鸡互动

- **点击小鸡** → 爱心表情 ❤️
- **按住不放 2 秒** → 省略号 + 逃跑 💬🏃
- **快速拖拽小鸡 1 秒** → 一团乱麻 😵
- **缓慢拖拽小鸡** → 移动位置

### 喂食 🌿

1. 右键系统托盘图标 → 点击 "🌿 喂食"
2. 干草会跟随你的鼠标移动
3. **左键** 放置干草 → 小鸡自动走过去吃
4. **右键** 取消喂食
5. 小鸡吃完会露出 😊 笑脸

### 系统托盘菜单

右键任务栏右下角的小鸡图标：

- **显示/隐藏** — 隐藏/显示小鸡
- **始终置顶** — 小鸡始终在所有窗口上方
- **鼠标穿透** — 开启后鼠标点击会穿透小鸡（只能通过托盘操作）
- **🌿 喂食** — 放干草喂小鸡
- **设置…** — 打开设置面板
- **退出** — 关闭桌宠

## ⚙️ 设置说明

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 小鸡大小 | 4x | 2x~6x 缩放（16×16 像素 → 32×32 ~ 96×96） |
| 动画速度 | 200ms | 走路帧切换间隔，越小越快 |
| 移动速度 | 3 | 走路速度（像素/帧），2~8 |
| 活跃度 | 5 | 1~10，越高走路越多、发呆越短 |
| 始终置顶 | ✅ | 小鸡始终在所有窗口上方 |
| 鼠标穿透 | ❌ | 开启后鼠标点不中小鸡 |
| 开机自启 | ❌ | 勾选后随 Windows 启动 |

> 所有设置在保存后**立即生效**，无需重启。

## ❓ 常见问题

### Q: 启动后看不到小鸡？
A: 看看任务栏右下角的系统托盘，如果有小鸡图标说明已启动。可能小鸡跑到了屏幕边缘，双击托盘图标或右键 → 显示/隐藏试试。

### Q: 点小鸡没反应？
A: 检查是否开启了"鼠标穿透"，如果是的话关掉它。如果小鸡在吃饭/逃跑中，也会暂时忽略点击。

### Q: 想卸载怎么办？
A: 退出程序，删除项目文件夹和桌面快捷方式即可（不写注册表，绿色无残留）。

### Q: 如何从 GitHub 更新？
A: `git pull` 拉取最新代码即可，`settings.json` 不会被覆盖。
