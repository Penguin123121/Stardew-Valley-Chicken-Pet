# 录制素材清单

## 录制参数

- 屏幕：2560×1600。
- Windows 显示缩放：150%。
- 最终裁切安全区：居中的 2560×1440。
- 最终输出：1920×1080、60 FPS。
- 录制工具：Windows 11 截图工具。
- 游戏标题界面如果无法录入系统声音，改用 Xbox Game Bar 单独录制。

## 每条素材通用要求

- 动作开始前和结束后各保留 3–5 秒。
- 关键内容不得贴近屏幕最上方或最下方 80 像素。
- 鼠标平稳移动，不快速乱晃。
- 不出现聊天通知、账号、真实姓名、Windows 用户名或私人文件。
- 失败镜头保留并以 `_02`、`_03` 后缀补录，不覆盖原文件。
